# Georgia Milestones Prep App

Production-ready starter codebase for a Georgia Milestones prep app using React, Vite, and Tailwind.

## Included
- Test simulation shell
- Question bank architecture
- Answer validation and scoring engine
- Navigator, flags, timer, and review state
- Modular file structure for future expansion

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Suggested next expansions
- Add subject-specific banks for math, science, and social studies
- Persist sessions with localStorage
- Add teacher/admin content ingestion
- Export score reports
